## Install dependencies

```sh
yarn
```

## 시작하기

```sh
yarn start
```

## Built With

- Language : Typescript
- Front-end : React, Styled Component, Storybook
- Back-end : Node.js, Express
- Test : Jest, Cypress, Enzyme, Supertest, nGrinder
- DevOps : Docker, Travis
